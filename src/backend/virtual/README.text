ГЕНЕРАТОР НАЗВАНИЙ ПРОЕКТОВ
============================

КАК ЗАПУСТИТЬ:

Если у вас WINDOWS:
1. Откройте папку "src"
2. Дважды щелкните по файлу "ЗАПУСК.bat"
3. Ждите завершения установки
4. Откройте браузер: http://localhost:3001/docs

Если у вас MAC или LINUX:
1. Откройте терминал
2. Перейдите в папку проекта:
   cd путь_к_проекту/src
3. Сделайте скрипт исполняемым:
   chmod +x start.sh
4. Запустите:
   ./start.sh
5. Откройте браузер: http://localhost:3001/docs

ЧТО УСТАНОВИТСЯ АВТОМАТИЧЕСКИ:
✅ Python зависимости (в виртуальное окружение)
✅ Ollama (если не установлен)
✅ Модель Qwen2.5:7b (4.6 ГБ)

ТРЕБОВАНИЯ:
- Python 3.8+
- 8+ ГБ оперативной памяти
- 10+ ГБ свободного места
- Интернет для первой установки

После запуска используйте API:
- Документация: http://localhost:3001/docs
- Пример запроса: POST /api/names
