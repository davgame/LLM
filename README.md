ГЕНЕРАТОР НАЗВАНИЙ ПРОЕКТОВ
============================

## КАК ЗАПУСТИТЬ:

Если у вас WINDOWS:
1. Откройте папку "src"
2. Дважды щелкните по файлу "start.bat"
3. Будет предложено установить модель Qwen2.5:7b
3. Ждите завершения установки
4. Откройте браузер: http://localhost:3001/docs

Если у вас MAC или LINUX:
1. Откройте терминал
2. Перейдите в папку проекта:
   cd путь_к_проекту/src
3. Сделайте скрипт исполняемым:
   chmod +x start.sh
4. Запустите:
   ./start.sh
5. Откройте браузер: http://localhost:3001/docs

ЧТО УСТАНОВИТСЯ АВТОМАТИЧЕСКИ:
✅ Python зависимости (в виртуальное окружение)
✅ Модель Qwen2.5:7b (4.6 ГБ)

ТРЕБОВАНИЯ:
- Python 3.8+
- 8+ ГБ оперативной памяти
- 10+ ГБ свободного места
- Интернет для первой установки

После запуска используйте API:
- Документация: http://localhost:3001/docs
- Пример запроса: POST /api/names
- Пример 1: Креативные названия для образовательной платформы
```
   {
  "text": "образовательная платформа для школьников с курсами по программированию",
  "smartMood": true
   }
```
- Пример 2: Простые названия для доставки еды
```
   {
  "text": "мобильное приложение для быстрой доставки еды из ресторанов",
  "smartMood": false
   }
```
Языковая модель для рекомендательной системы qwen2.5:7b

------------ Обучение + Проверка + Запуск = Qwen 2.5 + LoRa-адаптер=my-naming-model ------------
КАК ЗАПУСТИТЬ ОБУЧЕНИЕ LoRa-АДАПТЕРА:
1. Переходим ```cd src/backend```
2. Установите зависимости (если ещё не установлены)
   ```pip install torch transformers datasets peft accelerate bitsandbytes```
3. Запускаем файл ```train.py``` (создает адаптер)

ПОСЛЕ ОБУЧЕНИЯ:
1. Появится папка ```./qwen-lora-adapter/latest/``` с файлом ```adapter_model.bin``` (это LoRA-адаптер)
2. Создайте единую модель qwen 2.7 + LoRa командой: ```ollama create my-naming-model -f Modelfile```
3. Запуск сервера ```server.py```

ПРОВЕРКА:
1. После выполнения train.py: ls ```qwen-lora-adapter/latest/```
- Должны увидеть: adapter_model.bin
2. После создания единой модели qwen 2.7 + LoRa: ```ollama list```
- Должны увидеть: my-naming-model:latest
3. После запуска server.py откройте браузер и введите: http://localhost:3001/docs
